## 지원 범위

ROBOTIS는 AntBot의 **하드웨어 및 공식 제공 소프트웨어**에 대해 기술 지원을 제공합니다.

| 지원 포함 | 지원 미포함 |
| :--- | :--- |
| 하드웨어 결함 및 고장 | 사용자 작성 커스텀 소프트웨어 |
| 공식 펌웨어 및 업데이트 | 서드파티 센서·장치 연동 |
| 구동부 (Dynamixel 모터) | ROS 2 일반 사용 관련 질문 |
| 센서 하드웨어 (LiDAR, 카메라) | SLAM·Navigation 파라미터 튜닝 |
| 배터리 및 충전 시스템 | 비공식 OS·커널 변경으로 인한 문제 |
| RCU 보드 및 HMI 패널 | 네트워크 인프라 구성 |

:::caution
하드웨어를 **변형하거나 커스텀한 경우** 기술 지원 대상에서 제외됩니다. 여기에는 로봇 프레임 개조, 비공식 부품 교체, 전장 회로 변경 등이 포함됩니다.
:::

## 유료 기술 지원

기본 지원 범위를 넘어서는 전문 엔지니어링 서비스가 필요한 경우, 유료 기술 지원을 이용하실 수 있습니다.

| 항목 | 내용 |
| :--- | :--- |
| 원격 진단 | SSH를 통한 원격 점검 및 문제 해결 |
| 현장 방문 | 엔지니어 현장 출장 지원 |
| 커스텀 통합 자문 | 고객 환경에 맞춘 센서·시스템 연동 컨설팅 |

유료 지원에 관한 자세한 사항은 아래 연락처로 문의해 주십시오.

## 이슈 처리 절차

1. [알려진 이슈](/antbot/maintenance/known-issues/) 페이지에서 동일한 문제가 보고되어 있는지 확인합니다.
2. [에러 코드](/antbot/maintenance/error-codes/) 페이지에서 해당 에러 코드의 원인과 조치 방법을 확인합니다.
3. 위 내용으로 해결되지 않는 경우, 아래 방법으로 이슈를 접수해 주십시오.

### GitHub Issues

소프트웨어 버그 리포트 및 기능 제안은 GitHub Issues를 통해 접수할 수 있습니다.

- [Issue 등록](https://github.com/ROBOTIS-move/antbot/issues/new)

### 커뮤니티 리소스

ROS 2 일반 사용법, SLAM·Navigation 튜닝 등 지원 범위 외 소프트웨어 관련 질문은 아래 커뮤니티를 활용해 주십시오.

- [ROS Discourse](https://discourse.ros.org/)
- [ROBOTIS Community](https://community.robotis.com/)
