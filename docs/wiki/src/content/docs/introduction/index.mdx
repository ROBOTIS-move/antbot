**AntBot**은 ROBOTIS AI에서 개발한 오픈소스 **실외 자율주행 배송 로봇**입니다.

4개의 바퀴가 각각 독립적으로 조향하는 **스워브 드라이브(Swerve Drive)** 방식을 채택하여 전방향 이동과 제자리 회전이 가능하며, 라스트마일 배송 환경에 최적화되어 있습니다.

이 문서는 AntBot의 하드웨어 구성, 소프트웨어 아키텍처, 초기 설정부터 개발 가이드까지를 다룹니다. 로봇을 처음 접하는 엔지니어부터 시스템을 확장하려는 개발자까지, 필요한 정보를 빠르게 찾을 수 있도록 구성했습니다.

### 주요 특징

- **4WD 독립 조향**: 각 바퀴가 개별 조향하여 전방향 이동, 제자리 회전, 좁은 공간 주행이 가능
- **오픈소스 ROS 2 스택**: 제어기, 하드웨어 인터페이스, 센서 드라이버까지 전체 소프트웨어 공개
- **프로덕션 레디**: 실제 배송 현장에서 검증된 수준의 코드 품질
- **모듈형 설계**: 필요한 패키지만 선택하여 독립적으로 사용 가능
