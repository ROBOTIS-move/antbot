**AntBot** is an open-source **outdoor autonomous delivery robot** developed by ROBOTIS AI.

It uses a **Swerve Drive** system where each of the 4 wheels steers independently, enabling omnidirectional movement and in-place rotation, optimized for last-mile delivery.

This documentation covers everything from hardware configuration and software architecture to initial setup and developer guides. Whether you are an engineer deploying your first robot or a developer extending the system, you will find what you need here.

### Key Features

- **4WD Independent Steering**: Each wheel steers individually for omnidirectional movement, in-place rotation, and tight-space navigation
- **Open-Source ROS 2 Stack**: Full software stack including controllers, hardware interfaces, and sensor drivers
- **Production Ready**: Code quality validated in real-world delivery environments
- **Modular Design**: Use only the packages you need, independently
